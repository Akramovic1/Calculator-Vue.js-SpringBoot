# Calculator-Vue.js-SpringBoot

Simple Calculator using vue.js framework and SpringBoot.

## Installation

To use front-end Vue.js you should install npm firstly.

```bash
npm install
```

then you can run it by suing this command

```bash
npm run serve
```
